#  Ballistics-Calc

> Инженерный калькулятор для расчёта баллистических параметров снаряда, выпущенного из пускового механизма, с учётом реальных условий окружающей среды.

---

##  Описание

Программа предназначена для моделирования движения снаряда (толкание ядра рукой или иным пусковым механизмом) с возможностью учёта:

- сопротивления воздуха (с зависимостью от плотности, влажности, температуры и давления);
- силы трения в пусковом механизме;
- ветра (8 направлений + скорость).

Расчёт выполняется в два этапа:

1. **Аналитическая модель в вакууме** — для получения начального приближения.
2. **Численное интегрирование** (метод Рунге–Кутты 4-го порядка) с подбором начальной скорости методом бисекции.

---

##  Входные параметры

| Параметр      | Описание                                               |
| ------------- | ------------------------------------------------------ |
| `Sₓ`          | Экспериментальная дальность от основания механизма (м) |
| `α`           | Угол наклона вектора начальной скорости (градусы)      |
| `h₀`          | Высота установки механизма (рост до плеча) (м)         |
| `l`           | Длина пускового механизма (руки) (м)                   |
| `m`           | Масса снаряда (кг)                                     |
| `r`           | Радиус снаряда (м)                                     |
| `μ`           | Коэффициент трения в механизме (если есть)             |
| `T`           | Температура воздуха (°C)                               |
| `φ`           | Относительная влажность (%)                            |
| `p`           | Атмосферное давление (мм рт. ст.)                      |
| Ветер         | Направление (8 румбов) и скорость (м/с)                |
| Сопротивление | Да / Нет                                               |

---

##  Выходные данные

- Начальная скорость `v₀` (м/с)
- Время полёта (с)
- Максимальная высота (м)
- Скорость и угол при падении
- Сила запуска (Н)
- Потери механической энергии за полёт (Дж и %)

### Графики в терминале:

- Траектория `y(x)`
- Смещение от ветра `z(x)`
- Высота и дальность от времени
- Скорости от времени
- Энергии от времени
- Сила сопротивления от времени

##  Математическая модель

Полное описание вывода формул, уравнений движения и численных методов приведено в файле [`ballistics-calc.md`](./ballistics-calc.md)(для написания формул используется LaTeX).

### Кратко:

- **Без сопротивления** — аналитическое решение для дальности и начальной скорости.
- **С сопротивлением** — система ОДУ, решаемая методом Рунге–Кутты 4-го порядка.
- **Плотность воздуха** — по уравнению Клапейрона–Менделеева с поправкой на влажность (метод Ардена Бака).
- **Подбор `v₀`** — метод бисекции с целевой точностью `0.01` м по дальности.
